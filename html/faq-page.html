Frequently Asked Questions

What types of games can I host on your platform?

We support a wide range of game types, including first-person shooters, survival games, role-playing games, and more. Check our website for the full list of supported games.

How do I create a game server on your platform?

To create a game server on our platform, first, choose the game you want to host and then select the server plan that best fits your needs. Follow the instructions to set up your server and configure your game settings.

Can I customize the settings of my game server?

Yes, you can customize many settings for your game server, including the number of players, game rules, maps, and more. You can also install mods and plugins to enhance your gameplay experience.

How many players can join my game server?

The number of players that can join your game server will depend on the game and server plan you choose. We offer plans that support small servers for just a few players, as well as large servers that can host hundreds of players.

What kind of support do you offer?

We offer technical support for our game server hosting plans via email, live chat, and phone. We also have an online knowledge base with tutorials and guides to help you set up and manage your game server.

Can I switch to a different game or server plan after I've already set up my server?

Yes, you can upgrade or downgrade your server plan at any time. You can also switch to a different game if you want to try something new.

What payment methods do you accept?

We accept a variety of payment methods, including credit cards, PayPal, and cryptocurrency. Check our website for the full list of accepted payment methods.

How do you ensure the security of my game server and player data?

We have a variety of security measures in place to protect your game server and player data, including encryption, firewalls, and regular backups. We also monitor our servers 24/7 to ensure uptime and prevent unauthorized access.
